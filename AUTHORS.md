# Авторы / Authors

## Оригинальный проект / Original Project

**[averne](https://github.com/averne)** `<averne381@gmail.com>`
- Создатель и основной автор SwitchWave
- Creator and main author of SwitchWave
- Кастомный бэкенд декодирования FFmpeg для Tegra
- Кастомный графический бэкенд mpv на deko3d
- Кастомный аудио бэкенд через нативные API Nintendo
- Поддержка сетевых протоколов (Samba, NFS, SFTP, HTTP)
- Поддержка USB через libusbhsfs

## Зависимости / Dependencies

- [mpv](https://mpv.io/) — медиаплеер / media player
- [FFmpeg](https://ffmpeg.org/) — мультимедийный фреймворк / multimedia framework
- [deko3d](https://github.com/devkitPro/deko3d) — низкоуровневый GPU API для Switch
- [libnx](https://github.com/switchbrew/libnx) — библиотека homebrew для Switch
- [libusbhsfs](https://github.com/DarkMatterCore/libusbhsfs) — поддержка USB файловых систем
- [libsmb2](https://github.com/sahlberg/libsmb2) — SMB/Samba протокол
- [libnfs](https://github.com/sahlberg/libnfs) — NFS протокол
- [ImGui](https://github.com/ocornut/imgui) — немедленный UI фреймворк
- [ImPlot](https://github.com/epezent/implot) — библиотека графиков для ImGui
- [inih](https://github.com/benhoyt/inih) — INI парсер
- [libass](https://github.com/libass/libass) — рендеринг субтитров ASS/SSA
