## [Форк Dimasick-git / Dimasick-git Fork] — 2025-05

### Добавлено / Added

- **Русская локализация / Russian localization**
  - Компилируемая система i18n без накладных расходов в рантайме
  - Файлы: `src/i18n.hpp`, `src/i18n/ru.hpp`, `src/i18n/en.hpp`
  - Переведены все вкладки, кнопки, сообщения об ошибках, подсказки
  - Переведена статистика и меню плеера
  - Переведены сообщения консоли и справка

- **Автосборка CI / CI Auto-build** (`.github/workflows/build.yml`)
  - Сборка при каждом push и pull_request
  - Кеширование Docker-образа через GitHub Actions cache
  - Артефакт именуется с версией из Makefile и хешем коммита

- **Автоматические релизы / Auto-releases** (`.github/workflows/release.yml`)
  - Сравнивает `APP_VERSION` в Makefile с существующими git-тегами
  - При новой версии автоматически собирает и публикует релиз с артефактом
  - Описание релиза на русском и английском

- **Синхронизация с upstream / Upstream sync** (`.github/workflows/sync-upstream.yml`)
  - Еженедельная автосинхронизация с `averne/SwitchWave`
  - Защита кастомных файлов от перезаписи
  - Автоматическое создание Pull Request с изменениями
  - Возможность ручного запуска через `workflow_dispatch`

- **Документация / Documentation**
  - Полностью переписан README.md (EN вступление + RU основная часть)
  - Добавлен AUTHORS.md с информацией об авторах
  - Добавлен CHANGELOG.md (этот файл)

### Изменено / Changed

- Заголовки изменённых файлов обновлены с указанием Dimasick-git как соавтора
- `misc/mpv.conf`: приоритет русских аудиодорожек (`alang=ru,ja,en`) и субтитров (`slang=ru,ja,en`)
- `.github/workflows/build.yml`: обновлён до `actions/checkout@v4` и `upload-artifact@v4`, добавлена версия в имя артефакта

### Примечания / Notes

- Все изменения интерфейса совместимы с оригинальным бинарным форматом конфигурации
- Система i18n компилируется в compile-time, не влияет на производительность
- Для сборки с английским интерфейсом добавьте `-DSWITCHWAVE_LANG=0` в CXXFLAGS
